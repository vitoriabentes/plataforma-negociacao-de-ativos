# plataforma-negociacao-de-ativos
Plataforma distribuída para simulação de negociação de ativos financeiros, desenvolvida para a disciplina ACH2147 - Desenvolvimento de Sistemas de Informação Distribuídos (EACH-USP). Composta por microsserviços que gerenciam cadastro de ativos/investidores/corretoras, precificação em tempo real, ordens de compra/venda.
